# Dental Doctors Appointment Booking System (GPS-Based)

## Overview

This is a PHP-based web application that enables users to search for nearby dental clinics based on their location and book appointments according to available time slots.  

The system also provides appointment cancellation functionality, making the booking process flexible and user-friendly.

---

## Problem Statement

Manual appointment booking systems lack real-time availability tracking and location-based accessibility. Patients often struggle to find nearby dental clinics and manage appointments efficiently.

This project aims to provide a digital, location-aware solution for seamless appointment scheduling and management.

---

## Key Features

- Location-based clinic selection  
- Doctor listing under selected clinic  
- Slot-based appointment booking  
- Real-time availability checking  
- Appointment cancellation option  
- Patient registration and login system  
- Admin panel to manage clinics, doctors, and appointments  
- Database-driven backend using MySQL  

---

## Technologies Used

- PHP (Core backend logic)
- MySQL (Database management)
- HTML & CSS (Frontend design)
- JavaScript (Dynamic functionality)
- XAMPP (Local server environment)

---

## System Workflow

1. User registers and logs into the system  
2. User selects clinic based on location  
3. Available doctors and time slots are displayed  
4. User books appointment based on available slots  
5. Booking details are stored in the database  
6. User can cancel appointments if required  
7. Admin manages clinic, doctor, and appointment data  

---

## Database Structure

The system uses relational database tables such as:

- Users  
- Clinics  
- Doctors  
- Appointment Slots  
- Bookings  

CRUD operations are implemented for efficient data management.
